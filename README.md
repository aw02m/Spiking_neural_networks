# Spiking neural networks

## Install 
### Linux
```
git clone https://github.com/aw02m/Spiking_neural_networks
python3 -m venv env
source env/bin/activate
python setup.py install
```
### Windows
```
git clone https://github.com/aw02m/Spiking_neural_networks
python3 -m venv env
.\env\Scripts\activate
python setup.py install
```
